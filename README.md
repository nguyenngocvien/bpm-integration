
# 💳 BPM Integrations

Dịch vụ Spring Boot xử lý các API tích hợp hệ thống, xác thực dữ liệu đầu vào và quản lý lỗi tập trung.

---

## 🛠️ Công nghệ sử dụng

- Kế thừa integration trong System Data Toolkit


## ⚙️ Cài đặt và chạy

```bash
mvn install:install-file -Dfile=lib\integration.jar -DgroupId=teamworks -DartifactId=integration -Dversion=1.0  -Dpackaging=jar

```

```bash
mvn clean package

```
